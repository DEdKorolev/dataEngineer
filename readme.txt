Для запуска скрипта необходимо
1) в файле main.py указать реквизиты БД для подключения;
2) установить модули из файла requirements.txt;
3) версия python для стабильной работы указана в файле runtime.txt;
4) при проблемах с работой cx_Oracle попробовать шаги из рекомендаций по установке:
	https://cx-oracle.readthedocs.io/en/latest/user_guide/installation.html

Принцип работы со скриптом: 
После запуска скрипта сначала ожидаем отработку скрипта DDL(информация будет отображена в консоли).
Затем в папку files_in поочередно перекладываем файлы из папки files_out(или подобные файлы). 
После перекладывания одного файла через некоторое время формируется отчет в папке reports, а также появляется информация в консоли, что отработал скрипт Report.
Затем перекладываем следующий файл и так далее.
